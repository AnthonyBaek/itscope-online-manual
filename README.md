# ITSCOPE 온라인 매뉴얼

이 프로젝트는 ITSCOPE의 온라인 매뉴얼을 제공하는 웹 애플리케이션입니다.

## 주요 기능

- 온라인 매뉴얼 제공
- GitHub Pages를 통한 웹 호스팅
- 반응형 웹 디자인

## 기술 스택

- HTML5
- CSS3
- JavaScript

## 설치 및 실행

1. 저장소 클론
```bash
git clone [repository-url]
```

2. 웹 브라우저에서 `index.html` 파일 열기
